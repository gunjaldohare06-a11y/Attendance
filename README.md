# 📚 Attendance Management System

A modern **Attendance Management System** built using **Python Flask and SQLite**.  
This web application helps manage student records, mark attendance, and view attendance reports digitally.

---

## 🚀 Features

✨ Modern Animated UI  
✨ Add Student Details  
✨ Mark Daily Attendance  
✨ View Attendance Records  
✨ SQLite Database Integration  
✨ Responsive Design  
✨ Easy to Use Interface  

---

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Jinja2 Template Engine
